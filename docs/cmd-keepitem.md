# /keepitem

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/keepitem <item>
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
add/change the specified item in the .ini file to keep.
<!--cmd-desc-end-->

## Examples

- will set "roots" to be kept in the .ini  
  `/keepitem roots`
